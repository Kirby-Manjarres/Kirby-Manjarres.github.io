Spam & Ham Classifier

An AI-based tool that detects whether an SMS message is spam or ham (not spam). It uses NLP techniques like text cleaning and TF-IDF vectorization (with unigrams and bigrams) to convert text into meaningful numeric features. The logistic regression model is trained with balanced class weights to handle class imbalance, achieving around 98% accuracy. Results are visualized with confusion matrices and classification reports.
